## Projects

### [Scoopz](https://github.com/spacey-cadet/scoopz)
End-to-end TikTok automation pipeline running on Android via Termux. Watches a link inbox, downloads videos, extracts frames with `ffmpeg`, and generates captions via OpenAI. Includes a native Android share extension so you can queue links directly from the TikTok app — no server, no desktop, just a phone.

`Python` `Kotlin` `Android` `OpenAI API` `ffmpeg` `Termux`
